Blood Glucose Stats works by leveraging an open source project called Nightscout. Nightscout is connected to a continuous glucose monitor that I wear on my body to get current glucose values. Those values are stored in mongolab, and I use the api on dddiaz.com to query that database to update the glucose graphic in real time.

Github Link: https://github.com/dddiaz/Home

This is my personal website. It is build with Python, Flask, MongoDB, DynamoDB, and Docker. It is deployed on AWS using code pipeline to automatically build and deploy changes pushed to Github. It also runs my personal blood glucose API that returns real time blood glucose data.

I hope you enjoy checking out the source. Nothing is ever perfect, but I am proud of what I have accomplished here. If you notice anything out of place, feel free to let me know!

Github Link: https://github.com/dddiaz/Home

One of the major goals I had in mind when revamping my website is that I wanted continous deployment, so I could simply commit to github and have my changes automatically pushed to production.

To accomplish this I used AWS Code-Pipeline. AWS Code-Pipeline waits for any changes on github, and then as soon as it notices a change, it pulls the changes from github and rebuilds the docker image. After building the docker image it then deploys it to an ec2 micro instance using AWS Elastic Beanstalk.

Now I can make updates to my website without having to worry about the deployment process, which is truly a dream come true. :)
